# Flipkart Clone

A fully responsive Flipkart-inspired e-commerce frontend built using HTML, CSS, and JavaScript. This project replicates the core user interface and shopping experience of Flipkart, including product browsing, search, cart management, wishlist functionality, and checkout flow.

## Live Demo

https://gauravraj628.github.io/Flipkart_clone/

## Features

* Responsive Flipkart-style UI
* Product listing page
* Product details page
* Search functionality
* Shopping cart with quantity management
* Wishlist functionality
* Checkout page
* Product filtering and sorting
* Local Storage integration
* Mobile-friendly design
* Modern animations and hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Bootstrap 5
* Tailwind CSS
* Local Storage API
